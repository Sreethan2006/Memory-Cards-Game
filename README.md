<h1>Memory Cards Game</h1>

<hr>

<p>
A simple and fun memory-based card matching game where players flip cards to find and match pairs.
This project helps improve memory and concentration skills.
</p>

<hr>

<h2>How to Play</h2>
<ul>
  <li>Click on a card to flip it</li>
  <li>Flip another card to find its match</li>
  <li>If both cards match, they stay flipped</li>
  <li>If not, they flip back</li>
  <li>Continue until all pairs are matched</li>
</ul>

<hr>

<h2>Features</h2>
<ul>
  <li>Interactive user interface</li>
  <li>Card flip animations</li>
  <li>Match detection logic</li>
  <li>Beginner friendly code</li>
</ul>
