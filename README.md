Memory Cards Game

A simple and fun memory-based card matching game where players flip cards to find and match pairs. This project helps improve memory and concentration skills.

How to Play
Click on a card to flip it
Flip another card to find its match
If both cards match, they stay flipped
If not, they flip back
Continue until all pairs are matched

Features
Interactive UI
Card flip animations
Match detection logic
Simple and beginner-friendly code

Technologies Used

CSS
